Android nowadays use the Logical Volume Manager (LVM).

- The file system exposed via MTP is mounted under `/data/media/0` and `/storage/emulated/0`.
- `/storage/self/primary` is a symlink to `/storage/emulated/0`

```
alioth:/ $ mount
```
