QR 1 Whether this is a query (0) or an response (1).
Opcode 4 The type of query desired by the client. A few notable values are:
  • 0 a standard query (QUERY)
  • 1 an inverse query (IQUERY); this is currently obsolete but still functioning
  • 2 a server status request (STATUS)
  • 7-15 Unassigned

This authoritative definition of values in this field is maintained at https://www.iana.org/assignments/dns-parameters/dns-parameters.xhtml#dns-parameters-5.

A name server must copy this field as-is into its response.
